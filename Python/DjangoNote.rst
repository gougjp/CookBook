Django Note
===========================

安装
--------------

* 安装Python

* 安装django

.. code::

    pip install Django

创建项目
-----------------

.. code::

    django-admin startproject mysite

这行代码将会在当前目录下创建一个mysite目录, mysite目录下文件结构如下:

.. code::

    mysite/
    |-- manage.py
    `-- mysite
        |   `-- __init__.py
        |   `-- asgi.py
        |   `-- settings.py
        |   `-- urls.py
                    `-- wsgi.py

    1 directory, 6 files
